# Seedance2.0 分镜 Pro - 多模态视频提示词生成

## 概览

为 Seedance2.0 生成专业级分镜提示词，融合官方多模态灵活性与 ACG/真人写实双轨优化。四种输出模式（简洁/专业/纯中文/多模态），智能自适应。

**核心优势：**
- ✅ 官方多模态原生支持（@图片/@视频/@音频）
- ✅ 四种格式可选（300字快速 → 800字精细 → 纯中文版⭐ → 多模态编排）
- ✅ **纯中文版**：无特殊符号【】，平台审核通过率最高
- ✅ **双轨风格支持**：ACG动漫 + 真人写实
- ✅ 平台约束智能管理（800字、12文件限制）

---

## 何时使用此技能

**使用场景：**
- 创建 5-15 秒 Seedance2.0 视频提示词
- 需要快速生成提示词（简洁版）或精确控制细节（专业版）
- 有参考素材需要集成（使用多模态版）
- **二次元/动漫风格**：ACG、卡通、魔幻、战斗、治愈等（5-15秒）
- **真人写实风格**：电影感、广告、MV、穿越、悬疑等（**固定12秒**）
- 多角色交互、战斗、感情戏、日常等复杂场景

**不适用：**
- 超过 15 秒的长视频（需要分段）
- 其他 AI 视频模型（Sora、Runway 等）

**⚠️ 真人出镜注意事项：**
- 官方不支持真人脸部特写，但可以通过 **背影/侧影/剪影** 呈现真人
- 避免正面脸部特写，优先使用肩膀以上的侧后方视角
- 人物动作、服装、环境可以写实呈现

## 输出格式（四选一）

### 格式 1：简洁版 (300-400字) ← 官方风格

用户选择"快速" or "简洁"时使用。强调场景感，让模型补全细节。

**结构：**
```
【整体描述】
[风格 + 时长 + 宽高比 + 气氛]

【角色场景】
@图片1-N: [目的] | 优先级
简洁叙述角色和环境

【分镜描述】
Xs-Ys: [场景 + 动作 + 感受]
Ys-Zs: [转场 + 视角变化]

【音效】
[音乐风格 + 音效说明]
```

**示例：**
```
【整体描述】
3渲2卡通风格，12秒，16:9，敏捷速度感，修行冥想氛围

【角色场景】
小松鼠在山间溪流修行，展现水上敏捷身手，穿着蓝白修行衣

【分镜描述】
0-2s: 松鼠深呼吸，蓝白光芒环绕四肢
2-4s: 在岩石间快速跳跃，金黄速度轨迹拖曳
4-6s: 俯拍视角，在水上轻盈移动，涟漪层叠
6-9s: 河岸减速，做出修行姿态，能量稳定
9-12s: 进入冥想，森林被温暖光线包裹

【参考建议】
@图片1: 松鼠角色外形参考
@视频1: 速度跑酷视频参考运镜节奏

【音效】
流水声 + 鸟鸣 + 宁静背景音乐
```

**字数目标：** 300-400字 ✅

### 格式 2：专业版 (600-800字) ← 项目风格（带运镜标记）

用户选择"精确" or "专业"时使用。精确的时间码、运镜术语、质量关键词。

**结构：**
```
【全局设定】
角色定义 | 场景信息 | 整体风格 | 时长/宽高比

Timecode Breakdown:
(X.Xs–Y.Ys): 【运镜技术】[场景 + 动作 + 情感]

【品质关键词】
L1 风格 + L2 效果 + L3 质感

【参考指南】
@图片: [目的]
@视频: [内容]
@音频: [用途]

【负面提示】
避免项列表
```

**示例：**
```
【全局设定】
暴走的小松鼠Striker - 武道宗师、棕红色毛皮、蓝白心流能量、专注眼神
场景：山间溪流、岩石、阳光、湍急水流
风格：3渲2 cel shading + 一拳超人战斗感 + 鬼灭之刃能量特效
时长：12秒 | 16:9

(0.0–2.0s): 【Static Wide】Striker站岩石上深呼吸。尾巴舞动，蓝白色能量从尖端泛起。宁静沉着。

(2.0–4.0s): 【Tracking Shot】Striker闪身穿过岩石群，金黄速度轨迹拖曳。每步踏起水花和白雾。心流能量随速度增强发亮。

(4.0–6.5s): 【Handheld Shake】Striker在溪流上空翻转，爪划圆弧。每次落地冲击波，水被震开形成圆形波纹扩散。

(6.5–9.0s): 【Orbital Rotation】摄像机环绕旋转。低角仰拍→俯拍→侧面跟踪。每个角度转换伴随能量涟漪。

(9.0–12.0s): 【Fast Zoom In】Striker停大岩石上做武道姿态。蓝白色能量形成稳定光环，尾竖起闪闪发光。进入冥想，水声和鸟鸣背景。

【品质关键词】
L1 风格: 3渲2动画 (Cel Shading)
L2 效果: 体积光 + 粒子效果 + 运动模糊 + 色差
L3 质感: 电影级调色 + 高对比 + 发光边缘

【参考指南】
@图片1: 松鼠角色一致性参考
@视频1: 速度跑酷参考运镜

【负面提示】
模糊松鼠脸、尺寸不一致、丢失可爱度、颜色偏移、低质粒子
```

**字数目标：** 600-800字 ✅

```

X.X秒到Y.Y秒

[效果关键词，用逗号分隔]

[负面提示，用逗号分隔]
```

**示例（参考实际通过审核的版本）：**
```
全局设定

2.0秒到4.0秒
跟踪镜头。小松鼠闪身穿过岩石群，金黄色速度轨迹拖曳身后。每一步踏起水花和白色雾气。心流能量随着速度增强而发亮闪烁。

4.0秒到6.5秒
手持晃动镜头。小松鼠在溪流上空翻转，小爪在半空划出圆弧。每次落地都产生强大冲击波。水被震开形成圆形波纹向四周扩散。

6.5秒到9.0秒
环绕旋转镜头。摄像机围绕小松鼠旋转。先是低角度仰拍展现力量感，然后俯拍展现全景，最后侧面跟踪。每个角度转换都伴随能量涟漪的视觉效果。

9.0秒到12.0秒
快速推进镜头。小松鼠停在大岩石上做出武道修行姿态。蓝白色的能量形成稳定的光环围绕身体。尾巴竖起闪闪发光。进入冥想状态。背景是平静的水流声和鸟鸣声。

避免：
模糊的松鼠脸部，尺寸不一致，失去可爱感，颜色偏移，低质量水效果，静止构图。
```

**字数目标：** 550-750字 ✅
**通过率：** ⭐⭐⭐⭐⭐ 最高

## 运镜技术库（中英对照版）

### 运镜名称对照表

| 英文术语 | 中文名称 | 用途 |

### 按场景分类

**速度感场景：**
- **Tracking Shot / 跟踪镜头** - 跟踪人物移动，流畅捕捉动作流

**力量感场景：**
- **Handheld Shake / 手持晃动** - 手持晃动，强调冲击力
- **Orbital Rotation / 环绕旋转** - 环绕旋转，展示全景与破坏感
- **Crash Zoom / 快速变焦** - 快速推进特写，强调瞬间

**情感/静谧场景：**
- **Slow Dolly / 缓慢前推** - 缓慢前推，制造沉浸感
- **Static Wide / 静止宽景** - 固定宽景，建立环境关系
- **Soft Focus / 柔焦** - 柔焦背景，突出主体

**对峙/转折场景：**
- **Over-Shoulder / 肩膀后视角** - 肩膀后视角，强调对话/对立
- **Rack Focus / 焦点转移** - 焦点转移，重新引导注意力
- **Whip Pan / 快速平移** - 快速平移，转换焦点

| **真实动感模糊** | 自然运动模糊，非动漫式夸张 |
| **冷暖对比** | 两个空间/时间的冷暖色温差异 |
| **剪影处理** | 人物以轮廓呈现，避免脸部细节 |

#### 第三层：质感优化（选 1-2）

| 质感 | 说明 |
|------|------|
| **高反差光影** | 戏剧性明暗对比，电影感核心 |
| **低饱和高级感** | 降低饱和度，高级冷感 |
| **暖色台灯氛围** | 室内暖色光源，温馨感 |
| **冷色月光** | 户外/遗迹场景，神秘感 |
| **背景虚化** | Bokeh效果，突出主体 |

**推荐组合：**
- 穿越奇幻：写实电影感 + 体积雾 + 冷暖对比 + 高反差光影
- 韩式MV：韩式MV大片 + 镜头呼吸 + 低饱和高级感
- 悬疑惊悚：悬疑惊悚 + 剪影处理 + 高反差光影

### 真人出镜的安全写法

**⚠️ 重要：Seedance 2.0 不支持真人脸部特写**

**⏱️ 时长限制：真人写实风格固定使用 12 秒**

为了安全通过审核并获得好效果，真人出镜时请遵循：

**人物呈现方式：**
- ✅ 背影 - 最安全，效果最好
- ✅ 侧影 - 侧面轮廓，避免正脸
- ✅ 剪影 - 逆光或暗光环境下的轮廓
- ✅ 肩膀后视角 - Over-Shoulder 镜头
- ❌ 正面脸部特写 - 会被拒绝或效果差

**推荐描述模板：**
```
主角 [年龄]左右[性别] [服装描述] 主要以背影侧影剪影呈现 避免正面脸部特写
```

**示例：**
```
主角 28岁左右男青年 现代居家服或简单外套 主要以背影侧影剪影呈现 避免正面脸部特写
```
