## ✨ 核心功能说明

### 三种输出格式

#### 简洁版（300-400 字）
```
【整体描述】风格 + 时长 + 宽高比 + 气氛
【角色场景】简洁叙述
【分镜描述】Xs-Ys 时间段描述
【音效】音乐风格 + 音效说明
```
**优点：** ⚡ 快速、官方格式、即插即用
**缺点：** 👉 细节依赖模型补全

---

#### 专业版（600-800 字）
```
【全局设定】完整角色、场景、风格定义
(X.Xs–Y.Ys): 【运镜技术】场景 + 动作 + 情感
【品质关键词】L1+L2+L3 分层结构
【参考指南】@图片/@视频/@音频
【负面提示】避免项列表
```
**优点：** 🎬 精确控制、细节完整、质量有保证
**缺点：** 👉 字数多、交互轮次多

---

#### 多模态版（+参考素材）
```
【整体描述】简洁格式内容
【参考素材清单】按优先级组织（HIGH/MEDIUM/LOW）
【多模态提示词】融合参考的完整提示词
```
**优点：** 📸 充分利用官方能力、参考充分、专业
**缺点：** 👉 需要准备参考素材

---

### 运镜技术库

**速度感场景：**
- Tracking Shot - 跟踪，流畅捕捉
- Dynamic Angle Shift - 快速切角，紧张感
- Speed Line Effect - 速度线，加强快感

**力量感场景：**
- Handheld Shake - 手持晃动，冲击感
- Orbital Rotation - 环绕旋转，全景感
- Crash Zoom - 快速推进，强调瞬间

**情感/静谧场景：**
- Slow Dolly - 缓慢前推，沉浸感
- Static Wide - 固定宽景，环境关系
- Soft Focus - 柔焦背景，突出主体

**对峙/转折场景：**
- Over-Shoulder - 肩膀视角，对立感
- Rack Focus - 焦点转移，重新引导
- Whip Pan - 快速平移，转换焦点

### 平台约束
- **文本：** ≤ 800 字
- **图片：** ≤ 9 张
- **视频：** ≤ 3 个（≤15s 总长）
- **音频：** ≤ 3 个（≤15s 总长）
- **总文件：** ≤ 12 个

## 💡 常见问题

### Q: 新技能的简洁版效果会不会差？
A: 不会。简洁版遵循官方方法，Seedance2.0 模型会自动补全细节。通常效果与旧技能相当，但交互快得多。

### Q: 能否同时使用旧技能和新技能？
A: 可以。在过渡期两个技能会并行提供支持。建议逐步迁移到新技能。

### Q: 多模态版本真的支持 9 张图 + 3 个视频？
A: 是的。新技能会自动组织参考，确保不超过 12 文件限制。

### Q: 如何从简洁版升级到专业版？
A: 在新技能中直接更换输出格式选项，无需重新输入。技能会记住创意意图。
