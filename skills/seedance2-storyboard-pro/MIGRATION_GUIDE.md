# Seedance2.0 分镜技能：从 v1.0 到 Pro 的迁移指南

## 概览

**旧技能（seedance-storyboard-prompt）** ❌ 已废弃，但仍可用
- ✅ 强大的 ACG 深度优化
- ✅ 精确的时间码和运镜控制
- ⚠️ 字数限制压力（~800字）
- ⚠️ 缺乏原生多模态支持
- ⚠️ 交互流程较长（多轮对话）

**新技能（seedance2-storyboard-pro）** ✅ 推荐
- ✅ 官方多模态原生支持（@图片/@视频/@音频）
- ✅ 三种输出格式选择（快速/精确/多媒体）
- ✅ 平台约束智能管理
- ✅ 保留 ACG 深度优化
- ✅ 更快的交互流程

---

## 快速迁移路线图

### 场景 1：我想快速生成提示词

**之前（旧技能）：**
```
1. 输入想法

3. 获得 700-800 字分镜脚本

**现在（新技能）：**

```

**选择新技能的原因：** 快速生成，官方格式，字数轻松

**选择新技能的原因：** 参考素材处理更专业，多模态生成流程优化

流水声 + 鸟鸣 + 宁静背景音乐
```

#### 新技能的专业版（700字）
```
【全局设定】
暴走的小松鼠Striker - 武道宗师、棕红色毛皮...

(0.0–2.0s): 【Static Wide】Striker站岩石上深呼吸...

## 如何开始使用新技能

### 步骤 1：安装新技能

在 Claude Code 中注册新技能（如果是第一次）：

```bash
# 如果使用的是项目本地技能
# 则无需额外安装，直接使用 /seedance2-storyboard-pro
/seedance2-storyboard-pro
```

### 步骤 2：快速开始

**想要快速生成？**
```
用户：我想要一个松鼠修行的视频
技能：选择「快速」输出格式
输出：300-400 字简洁版，立即可用
```

**想要精确控制？**
```
用户：我需要精确到每一秒，角色是暴走松鼠...
技能：选择「精确」输出格式
输出：专业版，完全等同于旧技能
```

**有参考素材？**
```
用户：上传[图片][视频][音频]
技能：选择「多媒体」输出格式
输出：多模态版，自动优先级分类
```

### 步骤 3：生成后的流程

```
获得提示词
    ↓
复制到 Seedance2.0
    ↓
在文本框粘贴
    ↓
点击「生成视频」
    ↓
如不满意 → 返回技能调整参数
```

## 常见问题

### Q1：旧技能还能用吗？
**A:** 可以，但已标记为 deprecated。建议逐步迁移到新技能。两个技能会在过渡期并行提供支持。

### Q2：新技能的输出能直接粘贴到 Seedance2.0 吗？
**A:** 是的。所有三种格式（简洁/专业/多模态）都已验证可直接使用，无需任何转换。

### Q3：新技能的「简洁版」会不会效果不好？
**A:** 不会。简洁版遵循官方手册的自然语言方法，Seedance2.0 模型会自动补全细节。通常效果与旧技能相当，但交互更快。

### Q4：多模态版本真的支持 9 张图 + 3 个视频？
**A:** 是的。新技能会帮助你：
- 上传最多 9 张图片（角色、场景、风格等）
- 最多 3 个参考视频（运镜、转场、节奏）
- 最多 3 个音频文件（音乐、音效、节奏）
- 自动组织优先级，确保不超过 12 文件限制

### Q5：能否保留「负面提示词」功能？
**A:** 能的。新技能的专业版已包含负面提示词。简洁版也可以添加，用户只需要求即可。

### Q6：如何从「简洁版」升级到「专业版」？
**A:** 在新技能中直接更换输出格式选项，无需重新输入。技能会记住你的创意意图，只是改变输出方式。
